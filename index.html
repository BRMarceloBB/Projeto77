Projeto no console

O Visual Studio Code é um editor de código-fonte desenvolvido pela Microsoft para Windows, Linux e macOS. Ele inclui suporte para depuração, controle de versionamento Git incorporado, realce de sintaxe, complementação inteligente de código, snippets e refatoração de código.



O console. log é uma função que imprime um texto no console (no caso do JavaScript será possível visualizar através da ferramenta do desenvolvedor). Você utilizará esta função para te auxiliar no desenvolvimento, exibindo valores de variáveis em determinado ponto do código por exemplo (em tempo de execução).
